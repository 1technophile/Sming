12 /*
13  Protocol Description:
14  All in LSB first as it is sent. argo message array will be stored MSB first!
15  do LSB-MSB conversion in sendData
16  Byte 0: const 0 0 1 1 0 1 0 1
17  Byte 1: const 1 0 1 0 1 1 1 1
18  Byte 2: 0 0 0, 3bit Cool/Heat Mode, 2bit start SetTemp LSB first
19  Byte 3: 3bit End SetTemp, 2bit Fan Mode, 3bit RoomTemp LSB first
20  Byte 4: 2bit RoomTemp, 3bit Flap Mode, 3bit OnTimer
21  Byte 5: 8bit OnTimer
22  Byte 6: 8Bit OffTimer
23  Byte 7: 3bit OffTimer, 5bit Time
24  Byte 8: 6bit Time, 1bit Timer On/Off, 1bit Timer Program
25  Byte 9: 1bit Timer Program, 1bit Timer 1h, 1 bit Night Mode, 1bit Max Mode, 1bit Filter, 1bit on/off, 1bit const 0, 1bit iFeel
26  Byte 10: 2bit const 0 1, 6bit Checksum
27  Byte 11: 2bit Checksum
28 */
